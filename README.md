# 🌱 AgroSustentável - Agricultura Sustentável

Um site moderno e responsivo dedicado à promoção de práticas de agricultura sustentável e educação ambiental.

## 📋 Sobre o Projeto

AgroSustentável é uma plataforma web que apresenta informações completas sobre agricultura sustentável, incluindo:

- **Conceitos fundamentais** sobre sustentabilidade na agricultura
- **Práticas recomendadas** para cultivo responsável
- **Benefícios ambientais e econômicos**
- **Recursos educacionais** para agricultores e interessados

## 🎯 Objetivos

- Promover awareness sobre agricultura sustentável
- Fornecer informações acessíveis e de qualidade
- Conectar pessoas interessadas em práticas agrícolas responsáveis
- Contribuir para um futuro mais sustentável

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Design responsivo e moderno
- **JavaScript** - Interatividade e validação

## 🚀 Como Usar

### Visualizar o Site

1. Clone o repositório:
```bash
git clone https://github.com/vitorhugolealwall/futuro-sustent-vel-.git
cd futuro-sustent-vel-
```

2. Abra o arquivo `index.html` em seu navegador:
- Duplo clique no arquivo `index.html`, ou
- Use um servidor local (recomendado para melhor experiência)

### Usando um Servidor Local

Com Python 3:
```bash
python -m http.server 8000
```

Com Node.js (http-server):
```bash
npx http-server
```

Acesse `http://localhost:8000` no seu navegador.

## 📑 Seções do Site

### 1. **Navegação**
Menu sticky que acompanha o scroll com links para todas as seções principais.

### 2. **Hero Section**
Seção de introdução com chamada para ação destacada.

### 3. **Sobre Agricultura Sustentável**
Explica os três pilares:
- Responsabilidade Ambiental
- Viabilidade Econômica
- Responsabilidade Social

### 4. **Práticas Sustentáveis**
Apresenta 6 principais práticas:
- Rotação de Culturas
- Agroecologia
- Gestão de Água
- Compostagem
- Agricultura de Precisão
- Cultivo Orgânico

### 5. **Benefícios**
Lista 6 benefícios principais com checkmarks visuais.

### 6. **Contato**
Formulário para interessados entrarem em contato.
